# Public Assets

This folder contains static assets that are served directly.

## Logo File

Please download the SWY logo and place it here:

**File name:** `swy-logo.png`

**Source:** https://raw.githubusercontent.com/BDGOAL/Swyshopify/refs/heads/main/SWY-logo_new.png

**Steps:**
1. Download the logo from the GitHub link above
2. Save it as `/public/swy-logo.png`
3. The website will automatically load it from `/swy-logo.png`

## File Structure

```
/public
  ├── swy-logo.png  (you need to add this file)
  └── README.md     (this file)
```
